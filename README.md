# Rootkit
Rootkit based on ICMP packages


# Commands

Build and start module:
```
./run.sh
```

Delete module:
```
./stop.sh
```

Get prints dynamically:
```
dmesg -w | grep rootkit
```

Python shell example:
```
sudo python3 ./shell.py 45.11.26.23
```
